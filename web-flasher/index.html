AI Clock · ESP32-C3

仅适用于 4MB Flash 的 ESP32-C3。连接设备后，浏览器会先校验芯片家族，再分别写入启动、分区表和应用段。

日常升级请不要勾选“抹除”，这样会保留 Wi-Fi 和 LittleFS；只有首次安装或需要清空配置时才勾选。请先备份原固件，烧录期间不要拔线。烧录成功不代表屏幕引脚自动匹配；参考接线见项目文档。
当前浏览器不支持 Web Serial，请使用桌面版 Chrome 或 Edge。 请通过 HTTPS 或本机 localhost 打开此页面。

若自动进入下载模式失败：按住 BOOT，点按 RESET，松开 BOOT 后重试。ROM 下载模式不会被普通固件覆盖。
